> Mac Sublime Text 3 default key bindings:

CTRL + LEFT ARROW --> go to beginning of the line (MacOS), same as CTRL + A
CTRL + RIGHT ARROW --> go to end of the line (MacOS), same as CTRL + E
CMD + UP ARROW --> go to top of the file (MacOS)
CMD + DOWN ARROW --> go to bottom of the file (MacOS)

CMD + Z --> undo (MacOS)
CMD + F --> find (MacOS)

CMD + B --> open console and run (python) code
Right click on the console: see unsaved changes, see diff


CMD + P --> go to anything (shows full path of each open file)
CMD + P + @ --> go to functions in code, same as CMD + R
CMD + P + # --> go to variables in code
CMD + P + :15 --> go to line (15), same as CTRL + G


CMD + D --> select words to replace all at once (highlight first word occurrence, then click to select each subsequent occurrence)

CMD + click --> select multiple locations to add text

CMD + SHIFT + L --> multicursor selection of all lines to edit (highlight the desired subqsequent lines first, then apply the command and move cursor to desired location)

OPTION + LEFT CLICK --> to drag the multicursor on the left (beginning of the lines selected from clicking)

OPTION + CMD + 2 --> switch view to 2 columns (up to 4)
Move line - COMMAND + Control + Arrow Up or Down.
Duplicate line - COMMAND + Shift + D

CMD + K CMD + U --> convert to uppercase
CMD + K CMD + L --> convert to lowercase

ALT + CMD + [ --> code folding
ALT + CMD + ] --> code unfolding
